# Videos para Carrusel Hero

## Estructura de Videos
Colocar aquí los videos MP4 para el carrusel del Hero section. Cada video debe ser corto (10-15 segundos) y optimizado para web.

## Videos Requeridos
1. `hero-video-1.mp4` - Oficina y equipo trabajando
2. `hero-video-2.mp4` - Reuniones con clientes
3. `hero-video-3.mp4` - Procesos de trabajo contable
4. `hero-video-4.mp4` - Entorno empresarial

## Especificaciones Técnicas
- Formato: MP4 (H.264 codec)
- Resolución: 720p (1280x720) o 1080p (1920x1080)
- Duración: 10-15 segundos cada uno
- Tamaño: Máximo 5MB por video
- FPS: 30 fps
- Audio: Opcional (sin música de fondo para no interferir)

## Optimización
- Usar herramientas como HandBrake para comprimir
- Mantener calidad visual aceptable
- Considerar usar videos sin audio para mejor experiencia

## Uso en el Proyecto
Los videos se cargan automáticamente en el carrusel del Hero section. Se reproducen en bucle y cambian automáticamente cada 5 segundos.
